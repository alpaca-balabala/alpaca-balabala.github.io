数据类型的分类

数字

Python Number 数据类型用于存储数值。
数据类型是不允许改变的,这就意味着如果改变 Number 数据类型的值，将重新分配内存空间。

int

float

字符串

定义：它是一个有序的字符的集合，用于存储和表示基本的文本信息。

独特特性

  • 不可变性：字符串一旦创建，则不可以修改。一旦修改或者拼接，都会造成重新生成字符串，则要赋予一个新的值。

字符串格式化

1.格式化操作符：%

格式：%[(name)][flags][width].[precision]typecode
内涵：
    (name)      可选，用于选择指定的key
    flags          可选，可供选择的值有:
                 +       右对齐；正数前加正好，负数前加负号；
                 -        左对齐；正数前无符号，负数前加负号；
